# 🩺 Med-Assist AI: Clinical Support Assistant

Med-Assist AI is a multimodal decision-support tool designed to assist healthcare providers and patients by analyzing symptoms, vitals, and medical images. Powered by **Google Gemini 1.5 Flash**, it provides specialty-aware insights and automates clinical documentation.

---

## ✨ Features

* **Multimodal Analysis**: Upload images (rashes, throat, labs) alongside text descriptions for a more accurate assessment.
* **Clinical Specialty Routing**: Tailor AI behavior for General Practice, Pediatrics, Dermatology, and more.
* **Vitals Integration**: Input temperature, heart rate, and blood pressure to help the AI identify clinical abnormalities.
* **Automated SOAP Notes**: Generate professional Subjective, Objective, Assessment, and Plan notes with a single click.
* **Emergency Guardrails**: Built-in keyword detection to flag critical emergencies (e.g., chest pain, shortness of breath).
